# 🏇 Jockey Weight System — PWA

A Progressive Web App for managing jockey gear weights and calculating race-day setups.

---

## Deploy to Vercel (Free — 5 minutes)

### Step 1: Create a GitHub account
Go to https://github.com and sign up for free if you don't have one.

### Step 2: Create a new repository
1. Click the **+** button → **New repository**
2. Name it `jockey-weight-system`
3. Set it to **Public**
4. Click **Create repository**

### Step 3: Upload these files
1. On the repository page, click **uploading an existing file**
2. Drag ALL the files from this folder into the upload area:
   - `public/index.html`
   - `public/manifest.json`
   - `public/sw.js`
   - `public/icon-192.png`
   - `public/icon-512.png`
   - `vercel.json`
3. Click **Commit changes**

### Step 4: Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click **Add New Project**
3. Find your `jockey-weight-system` repository and click **Import**
4. Under **Root Directory**, type `public`
5. Click **Deploy**
6. Wait ~60 seconds — Vercel gives you a URL like `jockey-weight-system.vercel.app`

---

## Add to iPhone Home Screen

1. Open your Vercel URL in **Safari** on your iPhone
2. Tap the **Share button** (box with arrow) at the bottom
3. Tap **Add to Home Screen**
4. Name it **Jockey Weights** → tap **Add**

It will appear on your home screen and open full-screen like a real app.

---

## Features
- Jockey profiles with gear items and weights
- Saddle combo presets
- Auto-suggest optimal gear combination for target weight
- Lead bag calculator with loose lead amount
- Export/Import data as JSON backup
- Works offline once installed
- Data saves to device storage automatically

---

## Backup your data
Use the **📤 Export** button regularly to save a backup file.
Use **📥 Import** to restore on a new device.
